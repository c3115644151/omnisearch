# Omnisearch

A Minecraft mod that lets you search any item, block, or entity across all installed mods directly in-game, powered by mcmod.cn encyclopedia.

## Features

- **In-game search**: Search mcmod.cn without leaving the game
- **Item tooltip**: Hover over any item and hold TAB to search
- **Rich results**: View item details, sources, and mod info from mcmod.cn
- **Multi-version support**: Available for NeoForge (1.21.1+) and Forge (1.20.1)

## Installation

Download from [Modrinth](https://modrinth.com/mod/omnisearch) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/omnisearch).

## Building

```bash
./gradlew build
```

The built jar will be in `build/libs/`.
